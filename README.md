## Sobre o Workshop
Criaremos uma aplicação Full Stackcom uso de IA em todas as etapas:
* Definição dos Requisitos
* Implementação do Front-End
* Implementação do Back-End
* Integração
* Publicando no GitHub

## Tecnologias   
* React
* C# ASP.Net Core
* MongoDB
* Apis da OpenAI
* Apis da ElevenLabs
* ClaudeCode( Cursor como alternativa )
* MCP
* n8n
